Overview#

The library#

All functionality in magali is available from the base namespace of the magali package. This means that you can access all of them with a single import:

# magali is usually imported as mg
import magali as mg

import ensaio
import matplotlib.pyplot as plt

Load some data:

fname = ensaio.fetch_morroco_speleothem_qdm(version=1, file_format="matlab")
print(fname)
